### About ###
Swift/UIKit app for time measurment

#### Features: ####
   - Start/pause time count
   - Reset timer
   - Long press gesture on time label to reset timer
   - Time tracking in background after closing app 

#### Technology stack: ####
  - Swift/UIKit
  - Architecture MVC
  - Dependencies with CocoaPods
  - User Defaults 

### Installation ####
```sh
   cd .../Timer
   pod install 
```
